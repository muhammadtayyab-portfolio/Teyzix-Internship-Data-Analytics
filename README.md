## 📊 Task 1 — Customer Churn Analysis

**Dataset:** Telco Customer Churn (7,043 records)

End-to-end churn prediction project covering data cleaning, EDA, customer segmentation, and predictive modeling.

- **Models:** Logistic Regression (ROC-AUC 0.773, 82% Recall), Random Forest
- **Explainability:** SHAP for feature importance
- **Automation:** Weekly pipeline with Gmail SMTP email reporting
- **Dashboard:** Deployed on Streamlit Cloud

## 📊 Task 2 — Electronics Store Sales & Customer Behavior Analysis

**Dataset:** Self-generated UAE Electronics Store Sales (1,200 records, via Faker)

Full retail analytics workflow simulating a real-world business scenario — from synthetic dataset creation through cleaning, EDA, customer segmentation, and dashboard development.

- **Data Generation:** Faker-based synthetic dataset with realistic UAE cities, categories, and seasonal patterns
- **Cleaning:** Missing values, duplicates, inconsistent formats — fully documented
- **Segmentation:** RFM-based customer segmentation (High-Value, Loyal/Regular, At-Risk, Low-Engagement)
- **Dashboard:** Interactive Streamlit + Plotly dashboard with filters and KPIs
